Hexo 博客搭建指南

前置准备

  • 安装 Node.js (推荐版本 16 或以上)
  • 安装 Git
  • 确保已注册 GitHub 或 Gitee 账号

安装 Hexo

  1. 打开终端，安装 Hexo：
    npm install -g hexo-cli
  2. 创建博客文件夹并初始化：
    hexo init my-blog
  3. 进入目录并安装依赖：
    cd my-blog && npm install

运行本地服务

  • 生成静态文件：
    hexo generate 或 hexo g
  • 启动本地服务器：
    hexo server 或 hexo s
  • 访问：http://localhost:4000

部署到 GitHub Pages

  1. 安装部署工具：
    npm install hexo-deployer-git --save
  2. 修改配置文件 _config.yml，添加以下内容： 
    deploy:
  type: git
  repo: https://github.com/your-username/your-repo.git
  branch: gh-pages
  3. 执行部署命令：
    hexo deploy 或 hexo d

常见问题

  • 部署失败：确保 GitHub 仓库权限正确。
  • 本地无法运行：检查 node_modules 是否完整安装。
